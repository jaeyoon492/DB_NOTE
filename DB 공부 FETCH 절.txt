FETCH절

특정 집합을 출력시 출력하는 행의 수를 한정하는 역할을 한다. 부분 범위 처리시 사용된다.
--------------------------------------------------------------------------------------------------

FETCH절 문법

SELECT
	*

FROM
	TABLE_NAME

FETCH FIRST (N) ROW ONLY		//출력하는 행의 수를 지정한다. N을 입력하지 않고
;				//ROW ONLY 만 입력하면 단 한 건만 출력한다.
				//(괄호는 제외해야함_)

---------------------------------------------------------------------------------------------------

SELECT
	*

FROM
	TABLE_NAME

OFFSET N ROWS			//출력하는 행의수를 지정하면서 시작위치를 지정도 가능
FETCH FIRST (N) ROW ONLY		//OFFSET N값의 시작위치는 0이다. (괄호는 제외해야함_)
;

----------------------------------------------------------------------------------------------------

SELECT
	FILM_ID,
	TITLE

FROM
	 FILM

ORDER BY TITLE			//TITLE로 정렬한 집합 중에서
FETCH FIRST 4 ROW ONLY;		//4건의 행을 리턴 한다.
;

-----------------------------------------------------------------------------------------------------
