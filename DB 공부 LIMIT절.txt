LIMIT절

특정 집합을 출력 시 출력하는 행의 수를 한정하는 역할을 한다.
부분 범위 처리시 사용된다. PostgreSQL MySQL 등에서 지원한다.

LIMIT : n개 만 출력하라
OFFSET : 시작은 n 부터 해라

------------------------------------------------------------------------------------------

SELECT
	*

FROM
	TABLE_NAME

LIMIT N				//출력하는 행의 수를 지정한다.
;

-------------------------------------------------------------------------------------------

SELECT
	*

FROM
	TABLE_NAME

LIMIT N OFFSET M			//출력하는 행의 수를 지정하면서 시작위치를 지정한다.
;				//OFFSET M 값의 시작위치는 0이다.


---------------------------------------------------------------------------------------------
