WHERE 절은 집합을 가져올 때 어떤 집합을 가져올 것인지에 대한 (조건)을 설정하는 절이다.

WHERE 절 문법

SELECT
	COLUMN_1
	,COLUMN_2
	,중략...
FROM
	TABLE_NAME		//특정 테이블에서
WHERE	
	<조건>			//어떤 집합을 가져올지에 대한 조건을 준다.
;

----------------------------------------------------------------------------------------------------
조건종류

= : 같음
> : ~보다 큰
< : ~보다 작은
>= : ~보다 크거나 같은
<= : ~보다 작거나 같은
<>,!= : ~가 아닌
AND : 그리고
OR : 혹은

--------------------------------------------------------------------------------------------------------
AND절 사용예 

SELECT
	LAST_NAME
	,FIRST_NAME
FROM

	CUSTOMER

WHERE
	FIRST_NAME = 'Jamie'		//FIRST_NAME이 'JAMIE' 이면서
					// ''안에 내용은 대소문자까지 일치할것.
AND	
	LAST_NAME = 'Rice';		//LAST_NAME이 'Rice'인 행을 출력

----------------------------------------------------------------------------------------------------------

CTRL + SHIFT + F 누르면 자동으로 가독성 있게 바꿔준다.

----------------------------------------------------------------------------------------------------------
OR절 사용예

SELECT
	CUSTOMER_ID,
	AMOUNT,
	PATMENT_DATE
FROM
	PAYMENT
WHERE
	AMOUNT <= 1			//AMOUNT가 1이하 이거나
OR	AMOUNT >= 8;			//AMOUNT가 8이상인 행을 출력
